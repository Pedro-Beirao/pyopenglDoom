#  PyOpenGL Doom

https://www.youtube.com/watch?v=eVUoO2087no

___

### Description:

Doom renderer using PyOpenGL

### 3D Renderer:
```
python main.py path/to/doom.wad
```

### 2D Renderer: 
```
python main2d.py path/to/doom.wad
```


### Libraries

- Pygame
- PyOpenGL
- Numpy

### Goals

1. Read data from a WAD file
2. Draw the map in both 2d and 3d
